  Програма містить 2 вкладки: Main і Test. А також налаштування.

  Для налаштування бази голосних, приголосних букв а також символів, 
які додатково потрібно вилучити, чи не вилучати заходимо в 

File -> Characters.

Також існує пункт меню File -> Settings. 
Тут ми можемо налаштувати:
 - Separator - символ, яким будуть 
розбиватися склади
 - Separation Mode - c-cc та cc-c. 
Ц   е режими поділу на склади, у випадку, якщо в нас є 3 або більше 
приголосних і з них не було знайдено жодної
 висхідної послідовності. 
В 1 випадку ми ділимо після 1 приголосної. 
В 2 випадку - перед останньою
 - Characters Table - вибір таблиці 
сил приголосних.
 

Test - вкладка, призначена для моментального тестування якихось 
певних речень,слів, кусків тексту.


  Main:
В даній вкладці, перед 
натисканням кнопки процес, потрібно завантажити файл, який ми 
будемо процесати. Зробити це можна File -> Open. 

   Увага! Файл повинен бути у кодуванні UTF-8.
Після завантаження 
файлу - тиснемо кнопку "Process". Файл почне оброблятися, 
чекаємо доки прогрес бар дойде до кінця.

У випадку, якщо в процесі обробки слова, виникне помилка (наприклад 
знайдеться невідомий раніше символ) - це слово буде пропущено.

Інформація про помилку в певному слові виведеться в текстове поле 
Errors log.


   Після цього можемо зберегти результат на диск. Для цього потрібно 
зайти в File -> Save та вказати теку, в яку зберегти результат.

   Результат міститиме 4 файли: 
 - Syllables.txt - файл зі словами 
розбитими на склади
 - FirstSyllables.txt - лише перші склади
 - 
SyllablesCVV.txt - склади, перетворенні в формат CV 
(C - Consonant, V - Vowel)
 - SyllablesFirstCVV.txt - 
лише перші склади в форматі CV.
